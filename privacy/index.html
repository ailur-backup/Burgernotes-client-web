Information collected when signing up

When signing up for an account, we collect and store the following information:

  • Username, and your password hashed

    • This is used to authenticate you into our service when you log in

  • Date of account creation

    • This is used so that we can see inactive accounts and suspend them

  • Web browser "User Agent"

    • This is used so you are able to recognise your devices signing into your account

Information collected when logging in

When logging back in to your account, we collect and store the following information:

  • Web browser "User agent"

    • This is used so you are able to recognise your devices signing into your account

Information we collect while using our services

When you create a note, we collect and use this information:

  • Encrypted note content and title

    • This is used so that you may retrieve the note later

  • Note creator

    • This is used so that we can make sure your note is not given to anyone else

  • Note creation date

    • This is used so that your notes are listed by age if no edited date exists

  • Note last edited date

    • This is used so that your notes are listed by age

When you edit a note, we collect and use this information:

  • Encrypted note content and title

    • This is used so that you may retrieve the note later

  • Note last edited date

    • This is used so that your notes are listed by age

How we use your data

We use your data to make our services work. We don't share your information with third-parties.

We can't see the content and title of the notes you create

Your notes are encrypted end-to-end using AES-CBC (Advanced Encryption Standard - Cipher Block Chaining) 256-bit encryption.

We can only see:

  • Note creation date

    • This is used so that your notes are listed by age if no edited date exists

  • Note last edited date

    • This is used so that your notes are listed by age

  • Note creator

    • This is used so that we can make sure your note is not given to anyone else

Not note content or title.

Use of cookies

Burgernotes does not use cookies in the conventional sense. Burgernotes places infomation into localStorage, a type of blob storage similar to cookies, but not accessible to third-party websites or the server.

This ensures that a malicious server or third-party websites cannot tell your infomation.

  • DONOTSHARE-secretkey

    • This is used by the client to send to the server in order to identify you as... you.

    This contains no useful infomation and is simply a string of random bytes. If this is shared, you may simply remove the offending secretKey from your session list, by clicking "X" on the device you have logged in on to create this key.

  • DONOTSHARE-password

    • This is used by the aformentioned AES to encrypt and decrypt your notes. It never leaves your device.

    This is derived from your password and is therefore sensitive and should NEVER, EVER be shared. If you have done so, change your password as soon as possible.

  • SETTING-fontsize

    • This is used to determine the size of your font. It is not sensitive infomation and can be freely shared, though it never leaves your device.

  • FIRSTVISIT

    • This is used to determine if the user guide should be played, showing on a mobile device how touch gestures are used. It is not sensitive infomation and can be freely shared, though it never leaves your device.

  • NEWVERSION

  • This determines if you have seen the "What's New?" dialouge displayed in each new version. It is not sensitive infomation and can be freely shared, though it never leaves your device.

  • homeserver

    • This determines which server Burgernotes should connect to. It is not sensitive infomation and can be freely shared, though it never leaves your device.
